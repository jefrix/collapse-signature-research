# Stream Four Reference Catalogue

**Stream:** Four (Modern poetic and philosophical register)
**Status:** Scaffold – Cycle III, May 2026
**Maintainer:** All project tools (entry submissions); Jeff Ricks and Claude (curation)

## 1. Purpose

This catalogue is the project’s curated index of literary, philosophical, artistic, and other non‑empirical references that have been identified as **load‑bearing** for the book’s prose.

A reference is load‑bearing when it compresses an argument, supplies a structural metaphor, or stabilises a tonal shift that ordinary analytical prose cannot achieve without dilution. It is not decoration.

## 2. Scope

The catalogue includes:

- **Primary texts** (poems, novels, essays, plays, aphorisms, dialogues)
- **Philosophical works** (treatises, lectures, letters, concepts)
- **Visual art and music** (paintings, sculptures, compositions) – when they have been cited in a project document
- **Film, television, and oral tradition** – when cited
- **Scientific concepts used metaphorically** (e.g. Susskind’s holographic principle, black‑hole information paradox)
- **Any other cultural artifact** that has been proposed as a load‑bearing reference in a draft dispatch, chapter, or research note

The catalogue does **not** include:

- Raw empirical sources (those belong in Stream One/Two/Three ledgers)
- General bibliography entries not yet assessed for prose use
- Decorative allusions rejected by the editorial principle

## 3. Usage rules

1. **Every reference must earn its place.** When a tool proposes a reference, it must state the analytical work the reference performs. If the work is merely “adds gravitas,” the reference is rejected.
2. **A reference can be proposed by any project tool.** A Stream Three case brief might note a literary pattern; the same tool (or another) can file a catalogue entry giving the full bibliographic detail and a draft of the load‑bearing passage.
3. **All entries must be formatted according to `schema.md`.**
4. **Entries are living documents.** A reference may be updated if new analytical uses are found (e.g., a poem used for one argument later discovered to serve a second). Update the `analytical_use` field and date the revision.
5. **Jeff and Claude hold curatorial authority.** They can promote, demote, or retire entries based on editorial fit.

## 4. Integration with other streams

| Stream | How Stream Four connects |
|---|---|
| **Stream One** | Financial‑system dispatches may call for a literary or philosophical reference to compress a structural argument (e.g., a Magritte to illustrate the gap between asset prices and underlying collateral). References to be catalogued here. |
| **Stream Two** | Technocratic‑class mapping may draw on cultural texts the class itself cites (e.g., Heinlein, Rand). Those texts are catalogued here with notes on their political use. |
| **Stream Three** | Civilizational case briefs often identify literary‑strategic signals (e.g., Du Fu’s collapse poetry, Kraus’s satire). Those same works may serve double duty as Stream Four references. The catalogue cross‑links to case briefs. |

## 5. Editorial principles

From the foundation document:

> *Always ask whether the literary or philosophical reference is structural or decorative. Structural references do work the prose cannot do. Decorative references add gravitas without earning their place. … Be ruthless with this distinction. The reader will notice the difference whether or not they can articulate it.*

All catalogue entries must include a statement of the analytical work the reference performs. If the curator (Jeff or Claude) judges the work decorative, the entry is removed.

## 6. Directory structure

Entries are stored in sub‑folders by genre. The filename is `author‑short-title.md` in kebab‑case.

## 7. How to contribute

1. Identify a reference during research or writing.
2. Check the catalogue to see if it already exists. If it does, consider updating the existing entry.
3. Create a new file in the appropriate `entries/` sub‑folder using the `schema.md` template.
4. Fill in all fields. Leave no field blank; use `n/a` if genuinely inapplicable.
5. Commit the entry with a commit message: `Add reference: [Author], [Short Title]`.

## 8. Initial population

The catalogue is seeded with at least five entries drawn from materials already cited in Streams One–Three, to establish the format and ensure immediate usability.
